ABOUT

This project focuses on detecting deepfake images using both CNN and GAN models. The GAN model is trained and evaluated for authenticity prediction, while the CNN model is trained for binary classification of real or fake images. Data preparation, model construction, training, evaluation, and saving are covered for both models. Overall, the project provides a comprehensive approach to deepfake image detection using CNN and GAN models.
